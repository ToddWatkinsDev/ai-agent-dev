## Usage

### Command-Line Examples

```bash
# Check a single file
python file_integrity_checker.py check --file document.pdf

# Generate hash for multiple files
python file_integrity_checker.py hash --files file1.txt file2.txt file3.txt

# Check entire directory
python file_integrity_checker.py scan --directory ./data

# Verify against hash database
python file_integrity_checker.py verify --database hashes.json

# Compare two files
python file_integrity_checker.py compare --file1 original.iso --file2 backup.iso
```

### Python API

```python
from file_integrity_checker import FileChecker

# Create checker instance
checker = FileChecker()

# Generate hash
sha256_hash = checker.hash_file("document.pdf", algorithm="sha256")
print(f"SHA256: {sha256_hash}")

# Verify integrity
is_valid = checker.verify("document.pdf", "expected_hash_value")
if is_valid:
    print("File integrity verified!")
else:
    print("File integrity check failed!")

# Batch check
results = checker.batch_check([
    "file1.txt",
    "file2.txt",
    "file3.txt"
])
```

## Technical Details

### Supported Algorithms

- **MD5**: Quick hash, legacy support (128-bit)
- **SHA1**: Cryptographic hash (160-bit)
- **SHA256**: Cryptographic hash (256-bit) - Recommended
- **SHA512**: Cryptographic hash (512-bit)
- **CRC32**: Cyclic redundancy check (32-bit)

### Performance

- MD5: ~1000 MB/s
- SHA256: ~600 MB/s
- SHA512: ~400 MB/s
- CRC32: ~1500 MB/s
