## Usage

### Basic Widget Example

```python
from tkinter_ui import TkinterApp, Button, Label, Entry

app = TkinterApp(title="My Application")

label = Label(app, text="Enter your name:")
label.pack()

entry = Entry(app)
entry.pack()

button = Button(app, text="Submit", command=lambda: print(entry.get()))
button.pack()

app.run()
```

### Theme Management

```python
from tkinter_ui import TkinterApp, Theme

app = TkinterApp(title="Themed App")
app.set_theme(Theme.DARK)

# Switch theme at runtime
app.set_theme(Theme.LIGHT)
```

### Custom Forms

```python
from tkinter_ui import Form, FormField, FormValidator

form = Form(app)
form.add_field(FormField("email", type="email", required=True))
form.add_field(FormField("password", type="password", required=True))

if form.validate():
    data = form.get_data()
```

### Data Tables

```python
from tkinter_ui import DataTable

columns = ["ID", "Name", "Email", "Status"]
data = [
    [1, "John Doe", "john@example.com", "Active"],
    [2, "Jane Smith", "jane@example.com", "Inactive"],
]

table = DataTable(app, columns=columns, data=data)
table.pack(fill="both", expand=True)
```
