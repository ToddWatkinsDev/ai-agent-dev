## Decryption Algorithms

1. **AES (Advanced Encryption Standard)**
   - 256-bit key
   - Industry standard
   - Supports CBC, ECB, CTR modes

2. **RSA (Rivest-Shamir-Adleman)**
   - 2048-bit key
   - Public key cryptography
   - Requires corresponding private key

3. **DES (Data Encryption Standard)**
   - Legacy support
   - 56-bit key
   - Not recommended for new data

4. **Blowfish**
   - 64-128 bit block size
   - Fast decryption
   - Useful for file decryption
