## Usage

### Basic Color Selection

1. Launch the application
2. Use the RGB sliders to select your desired color
3. View the color preview in the main display area
4. Check the color values in RGB, HEX, and HSV formats

### Color Format Conversion

```python
from rgb_picker import ColorConverter

# RGB to HEX
converter = ColorConverter(255, 128, 0)
print(converter.to_hex())  # Output: #FF8000

# HEX to RGB
rgb = ColorConverter.from_hex("#FF8000")
print(rgb.values)  # Output: (255, 128, 0)
```
