## Encryption Algorithms

1. **AES (Advanced Encryption Standard)**
   - 256-bit key
   - Industry standard
   - Very secure

2. **RSA (Rivest-Shamir-Adleman)**
   - 2048-bit key
   - Public key cryptography
   - Slow but very secure

3. **DES (Data Encryption Standard)**
   - Legacy support
   - 56-bit key (weak, not recommended)

4. **Blowfish**
   - 64-128 bit block size
   - Fast encryption
   - Useful for file encryption
